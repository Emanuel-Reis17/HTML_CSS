Formatação de textos

Podemos fazer uma marcação no texto usando a tag "mark"

Para não ficar tão feio, também podemos colorir a marcação usando a tag "style="background-color: ;"

Ou também. podemos usar um seletor em "style mark{}"

Sublinhar e Deletar

Podemos riscar um texto usando a tag "del"

Podemos também sublinhar um texto usando a tag "ins"
